# QuestDo

## 🧠 Idea

QuestDo ist eine gamifizierte Challenge-Plattform, bei der Nutzer reale Aufgaben („Quests“) erledigen, Punkte sammeln und ihren Fortschritt mit Freunden verfolgen.

Fokus:

* Motivation durch Gamification
* Aktionen im echten Leben
* Einfache soziale Konkurrenz



## 🎯 Ziel

Alltägliche Aufgaben interessanter machen, indem sie in kleine Challenges verwandelt werden.

Beispiele:

* Sport & Fitness
* Lernaufgaben
* Kleine tägliche Herausforderungen



## 🔁 Kern-Loop

1. Quest erhalten oder auswählen
2. Im echten Leben erledigen
3. Als erledigt markieren
4. Punkte erhalten
5. Neue Quests freischalten



## 🧩 Quests

* Manuell erstellt (Admin / Nutzer)
* Nutzer können Ideen einreichen



## 🏆 Motivation

* Punkte pro Quest
* Level-System
* Einfache Rangliste (Leaderboard)
* Optionale Streaks



## 🌐 Plattform

* Zuerst Web-App
* Später optional als PWA



## 🛠️ Technik

* Frontend: HTML / CSS / JavaScript
* Backend: Flask
* Datenbank: SQLite



## 🚀 MVP

Muss enthalten:

* Nutzer
* Quest-Liste
* „Quest erledigt“-Button
* Punktesystem + Streaks
* Einfache Rangliste
