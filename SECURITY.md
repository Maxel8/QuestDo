# Security Policy

## Supported Versions
This project is under active development.

Only the latest version (main branch / latest release) is supported with security updates.

Older versions are not maintained.


## Reporting a Vulnerability

If you discover a security vulnerability, please report it using GitHub Security Advisories:

https://github.com/Maxel8/QuestDo/security/advisories/new

We aim to respond within a few days.

If the issue is confirmed, we will fix it as soon as possible and coordinate disclosure if needed.
