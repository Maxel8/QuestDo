Über QuestDo

QuestDo gibt Dir und Deinen Freunden kleine Aufgaben (Sidequests) zum Erledigen. Mit jeder erledigten Quest erhält man XP, um im Leaderboard aufzusteigen.

Mitwirken

Du hast nicht nur an der Nutzung von QuestDo Interesse, sondern willst aktiv helfen, dass Projekt weiterzuentwickeln? Dann schau in dem GitHub Repository vorbei.

Möglichkeiten zur Unterstützung:

  • 💻 Code verbessern
  • ✨ neue Features entwickeln
  • 🐛 Fehler melden
  • 📜 Quests vorschlagen

Weitere Möglichkeiten sind in Arbeit (wie Übersetzungen), außerdem sind wir auch für Deine zusätzlichen Ideen offen.
